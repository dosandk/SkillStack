# Multiple skill selection in one run

## Summary

Allow the user to install several skills from the same repository in a single command
invocation, applying the chosen platforms to all selected skills.

## Unit test requirements

- Deduplication logic: given a list of selected skill identifiers containing
  duplicates, produces the deduplicated set used for install, as a pure function.
- Batch result aggregation logic: given per-skill/per-platform outcomes
  (success/failure), computes the correct overall exit code and summary structure.

## Integration test requirements

- Select three skills and two platforms against a fixture repo and a temp project
  directory — assert all combinations are installed and a summary lists each
  skill/platform result.
- Simulate one skill failing during a batch install (a forced write failure in the temp
  directory) — assert the remaining skills still install, the failed one is clearly
  reported, and the command exit code reflects a partial failure.
- Select the same skill twice in one CLI invocation — assert it is installed once per
  platform in the temp project directory with no duplicate writes.
