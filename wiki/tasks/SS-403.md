# Show live GitHub metadata (stars) in results

## Summary

Augment search results and detail views with up-to-date GitHub metadata (such as star
count) fetched live via the backend (SS-513) and merged with the stored Firestore data.

## Unit test requirements

- View-model merge logic combines stored fields (validation/commit data) with live fields (stars) into one display object, keeping each field's source consistent with its type
- Merge logic omits or marks live fields as "last known"/stale when live metadata is absent, without altering stored fields

## Integration test requirements

- Render a result/detail view with a mocked API client returning both stored and live GitHub metadata, and assert star count and other live fields display alongside stored data
- Render a result/detail view with a mocked API client simulating a GitHub-unavailable/rate-limited response, and assert stored data still renders, live metadata degrades gracefully, and no error/page crash occurs
