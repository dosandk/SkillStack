# List of user's uploaded skills/repos

## Summary

On the profile page, list the repositories/skills the user has uploaded with each
entry's current validation status, providing quick access to manage them.

## Unit test requirements

- List component renders each uploaded repo/skill entry with its current validation
  status and a link to its detail/validate view, given a populated array
- List component renders an empty-state prompt to upload when given an empty array,
  without erroring
- List component correctly displays the range of validation status values (e.g.
  pending, passed, failed)

## Integration test requirements

- Rendering the profile page with a mocked SS-433 query response containing the
  caller's repos displays the full list with correct statuses and working detail links
- Rendering the profile page with a mocked SS-433 response returning zero repos displays
  the empty-state prompt with no error
- Rendering the profile page only displays entries returned by the SS-433 query for the
  current user (boundary enforcement itself is verified by SS-433's own integration
  tests)
