# Platform target selection + file placement

## Summary

After skill(s) are selected (SS-501) and confirmed, prompt for target platform(s) —
Claude, Cursor, Copilot, multi-select ([FR-CLI8](../architecture.md#fr-cli8)) — and, once file content is available
(SS-503), write each selected skill's directory into the correct folder per selected
platform.

## Unit test requirements

- The platform prompt accepts one or more selections.
- Given a skill's file set and a set of selected platforms, the write step places that
  skill's full directory (including all subdirectories) under each platform's expected
  target folder.
- Selecting zero platforms is rejected with a clear message, not a silent no-op.

## Integration test requirements

- Running the full flow with two platforms selected and one skill produces the correct
  files under both platform target folders on a temp filesystem.
