# Feature: Catalog — Browse & Filter Skills

## Description

The public front-end catalog that lists skills as cards with search, tag/agent
filtering, and time-windowed sort tabs (Trending / Hot / All Time), plus status and
install-command affordances. This is the existing, shipped browsing experience that
the search-discovery feature extends once live Firestore/GitHub data is available.
